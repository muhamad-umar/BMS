## 🚀 About The Project

SmartStock is a fully responsive, enterprise-grade Business Management System (BMS) designed to handle inventory, sales, customer ledgers, and profit analytics. Built with a focus on speed, offline-capability, and beautiful UI/UX, the application provides an app-like experience on both desktop and mobile devices.

### ✨ Key Features
- **📦 Inventory Management:** Track stock movements (In/Out), categorize products, and view individual stock history.
- **💰 Sales & POS:** Record sales, apply discounts, and manage customer balances dynamically.
- **👥 Customer Ledgers:** Maintain detailed customer profiles with running balances and individual transaction ledgers.
- **📈 Profit & Analytics:** Dedicated, role-protected profit breakdowns and real-time team activity dashboards.
- **📱 PWA Ready:** Installable as a native app on iOS and Android with aggressive service worker caching for instant load times.
- **🔒 Secure Architecture:** Robust Role-Based Access Control (RBAC), Row Level Security (RLS) policies, and encrypted auth flows.

## 🛠️ Technology Stack

SmartStock leverages a modern, lightweight, and incredibly fast tech stack:

### Frontend Architecture
| Technology | Purpose |
| :--- | :--- |
| **HTML5 & CSS3** | Semantic structure with a custom, fluid, and responsive design system (no CSS frameworks bloat). |
| **Vanilla JavaScript (ES6)** | Pure, module-based JavaScript for DOM manipulation and state management. |
| **PWA (Service Workers)** | Offline capabilities, cache-busting, and native "Add to Home Screen" support. |

### Backend & Database (BaaS)
| Technology | Purpose |
| :--- | :--- |
| **Supabase** | The core backend provider handling real-time database and authentication. |
| **PostgreSQL** | Relational database utilizing triggers, views, and RPCs for complex backend calculations. |
| **Deno Edge Functions** | Secure, serverless functions (e.g., `admin-create-user`) for privileged administrative tasks. |

### Tooling & UI Assets
| Technology | Purpose |
| :--- | :--- |
| **Vite** | Next-generation frontend tooling for ultra-fast HMR and optimized production builds. |
| **FontAwesome 6** | Scalable vector icons for a premium user interface. |
| **Google Fonts** | Utilizing **Inter** for crisp, highly readable, modern typography. |
